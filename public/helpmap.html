Info zur Karte

Auf der Karte hat jede Mess-Station eine Säule, deren Höhe dem Messwert des PM10-Feinstaubwertes entspricht. So lange der Messwert < 30µg/m3 ist, ist die Säule grün, zwischen 30 und 50µg/m3 ist sie orange, über 50µg/m3 wird sie rot und über 100µg/m3 violett. Bei Werten über 100 wird die Säule nicht höher um das Bild nicht zu sehr zu überladen. D.h. also Werte > 100µg/m3 können nicht mehr an der Höhe unterschieden werden.
Wenn der Sensor seit mind. einer Stunde keine Werte gesendet hat, wird seine Säule als schwarze Scheibe dargestellt. Sensoren, die sich länger als eine Woche nicht am Server gemeldet haben, werden nicht mehr angezeigt. Auch fehlerhafte Sensoren werden nicht angezeigt (siehe 'Fehlerhafte Sensoren')

Die Karte wird so ausgerichtet, dass der zuletzt in der Grafik angezeigte Sensor in der Mitte erscheint. Zum leichteren Auffinden dieses Sensors wir er komplett in weiß dargestellt.

Ein Klick auf eine Säule zeigt eine kleine Anzeigetafel mit den aktuellen Werten für PM10 und PM2,5 (jeweils den Mittelwert der letzten 5 Minuten). Außerdem wird die Nummer des Sensors angezeigt sowie ein Link. Bei einem Klick auf den Link wird der zeitliche Verlauf dieses Sensors als Grafik angezeigt.

Die Karte kann durch Klick auf das Plus- oder Minus-Zeichen rechts unten gezoomed werden. Durch Klicken und Festhalten mit der Maus an beliebiger Stelle kann die Karte verschoben werden.

Mit dem Zentrieren-Knopf kann ein neuer Karten-Mittelpunkt festgelegt werde. Es muss eine Ort eingegeben werden. Dieser neue Mittelpunkt wird nicht gespeichert.

Mit dem Knopf Verkehr einblenden wird die Google-Verkehrsinformation mit in die Karte eingeblendet. Staustrecken werden rot markiert. Ein erneuter Klick auf diesen Knopf blendet den Verkehr wieder aus

Mit den zurück-Knopf gelangt man zurück zur zuletzt angezeigten Linien-Grafik.

Die Daten werde alle 5min vom Server geholt und dann neu gezeichnet. Falls es zu Übertagungsfehlern kommt, bleiben die alten Werte erhalten. Unter der Karte steht das Datum und die Uhrzeit der angezeigten Werte.

Die Grafik eines Sensors kann auch direkt aufgerufen werden durch feinstaub.rexfue.de/SNR mit SNR = der Sensornummer ( Beispiel: feinstaub.rexfue.de/140 )

Die Daten aller Sensoren können auch über eine API direkt abgeholt werden. Die Beschreibung der Api ist unter Feinstaub_API.pdf zu finden. /p>
